# nixGLBurnet

### Device 

Google Burnet/Esche (MT8183) Board

Only tested on a form of Debian called [Velvet OS](https://github.com/velvet-os/imagebuilder/blob/main/systems/chromebook_kukui/readme.md), specifically steered towards supporting chromebooks with linux.

### Fastfetch:
Related information:
- Kernel: Linux 6.12.50-stb-cbm+
- CPU: mt8183 (8) @ 1.99 GHz
- GPU: Mediatek mt8183b-mali [Integrated]
